## Quickstart
```
git clone http...
cd prisma-auth0-vue/client
cp example.auth0-variables.js auth0-variables.js // Then fill in your auth0 params
yarn install
yarn dev
cd ../sever
cp example.env .env // Then fill in your PRISMA/AUTH params
yarn install
yarn dev
```
